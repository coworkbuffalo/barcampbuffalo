# BarcampBuffalo

Quick little Rails app for BarCampBuffalo 2012.

## Setup

*Note:* Requires PostgreSQL.

Run the following:

    rbenv install 1.9.3-p194
    ln -s ~/Dev/barcampbuffalo ~/.pow/barcampbuffalo
    ./script/setup

Open it up!

    open http://barcampbuffalo.dev
